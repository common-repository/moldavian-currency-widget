== Description ==

A simple plugin that creates widget with exchange rates of moldavian leu in relation to other currencies. Uses rates of Banca Nationala a Moldovei. You can choose necessary currencies and language of displayed currencies.

== Installation ==

1. Upload unarhived `mdl-currency.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place Moldavian Currency Widget in sidebar.
